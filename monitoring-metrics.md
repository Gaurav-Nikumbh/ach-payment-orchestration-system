# Monitoring Metrics

Payment systems require continuous operational monitoring.

Key metrics include:

## Reliability Metrics

- transaction success rate
- processor latency
- error rates

---

## Queue Metrics

- retry queue backlog
- batch processing delays
- job execution failures

---

## Compliance Metrics

- return code distribution
- cutoff window violations
- reconciliation discrepancies

Monitoring these signals helps detect issues early and maintain platform reliability.
